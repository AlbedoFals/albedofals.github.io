Markdown区块元素总结

1. 段落和换行

Markdown中段落的形成，必须前后有一个以上空行。如果需要在段落内部换行，需要插入换行符，通常需要先按两个以上的空格然后回车。

2. 标题

Markdown有两种标题语法，类Setext和类atx形式。

4. 列表

Markdown支持无序列表和有序列表。

无序列表使用星号、加号或者减号作为列表标记:

  • Red
  • Green
  • Yellow

等同于：

  • Red
  • Green
  • Yellow

也等同于：

  • Red
  • Green
  • Yellow

有序列表使用数字加上英文句点（数字.）：

  1. Google
  2. Alibaba
  3. Apple

由于有序列表上使用数字的顺序并不会影响HTML结果，下面两种情况得到的结果完全一致。

第一种：

  1. Google
  2. Alibaba
  3. Apple

第二种：

  1. Google
  2. Alibaba
  3. Apple

列表项目（无论有序列表还是无序列表）可以包含多个段落，每个项目下的段落必须缩进4个空格或者1个制表符：

6.分割线

你可以在一行中用3个以上的星号、减号、底线来建立一个分隔线，此时行内不能有其他东西。另外，星号或者减号中间插入空格，也是可以的。
